> Requirements
- A rooted device with zygisk enabled
- A `meta-module` if you're not using magisk
#

> Usage
- Enable magisk's `built-in Zygisk` or flash `ReZygisk` or `ZygiskNext` or `NeoZygisk`
- Clear data of `google photos` app
- Flash this module
- Reboot your device
